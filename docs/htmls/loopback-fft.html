Browser loopback scope + FFT null test

Plays a bin‑snapped sine out the chosen DAC and captures it on the chosen ADC (loopback cable DAC→ADC). Input and output run in separate AudioContexts so each stays at its own native rate. A pool of Web Workers computes the FFTs in parallel; overlap processing runs more frames per second. With FLL + coherent + reject on, a frequency‑lock loop steers the generator onto the captured tone, glitch frames are rejected, and averaging goes coherent (each frame de‑rotated to a common time origin) — the skirt collapses and the floor digs below the incoherent level. Needs a secure context (https or localhost).
