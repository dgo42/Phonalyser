Phonalyser

Прецизійний стенд для вимірювання аудіо: генератор сигналів, осцилограф, FFT-аналізатор.

Натисніть F1 будь-де в програмі, щоб відкрити цю довідку. Натисніть Ctrl+F1, щоб перейти безпосередньо до розділу, що стосується елемента керування, який зараз у фокусі.

🚀 Найкоротший у світі HOW-TO — повне вимірювання THD з точністю до частин на мільярд, від калібрування до результату, на одному екрані. Почніть звідси.

🔍 «Покажчик і пошук» — пошук по всьому посібнику або перегляд алфавітного списку тем і термінів.

💡 Поради — дрібниці, які мають велике значення; також відображаються по одній під час запуску.

🙏 Подяки та визнання — натхнення і вдячність.

Модулі

Генератор сигналів — генерація тонів, розгортка, шум, відтворення WAV-файлу Осцилограф — вигляд у часовій області в реальному часі, тригер, таблиця вимірювань FFT-аналізатор — спектр, гармоніки, THD / SNR / ENOB Частотна характеристика — розгорнута амплітуда & фаза, еталон RIAA / IEC, режим порівняння Налаштування — аудіо-бекенд і пристрої, вигляд осцилографа та FFT

Теорія роботи

Теорія роботи — як прилад працює зсередини: аудіо-бекенд, кільцевий буфер & споживачі, DDS-генератор сигналів

Інструменти та стенд

Вимірювальний стенд і проводка — клітка Фарадея, кручена пара CAT.8 з XLR, заземлення в одній точці та бюджет наведеного шуму

Компонування

Головне вікно розділено на панель Генератора сигналів ліворуч (ширина регулюється — перетягніть роздільник або задайте у Налаштуваннях) і вертикальний роздільник між Осцилографом (вгорі) та FFT-аналізатором (внизу) праворуч. Кожну панель можна згорнути до смужки заголовка подвійним клацанням по заголовку; повторне подвійне клацання відновлює її.

Кожен модуль має власну панель інструментів — смужку вкладок налаштувань нижче основного вигляду з вкладками для параметрів модуля. Смужку вкладок також можна згорнути (подвійне клацання по смужці), щоб звільнити вертикальний простір для видимого графіка.

Угоди

  • Крокові селектори — поля V/div, t/div, FFT length мають маленькі стрілки вгору/вниз. Колесо миші прокручує стандартні значення (1-2-5-10), а введене вручну значення приймається, якщо воно має сенс для даного поля.
  • Числові поля — приймають введені значення та крокове прокручування колесом миші. Крок колеса залежить від поля (5 % для частоти, ±1 одиниця для номерів гармонік тощо).
  • Плитки вкладок — маленькі значки, намальовані під кожною міткою вкладки, показують активне значення для найважливіших елементів керування цієї вкладки з першого погляду, навіть коли тіло вкладки згорнуто.
  • Перехрестя — наведення курсора всередині графіка показує пунктирний хрест з плаваючим інформаційним блоком поруч із курсором.
