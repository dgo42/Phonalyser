Налаштування

Ця сторінка описує параметри налаштувань. Про те, чому аудіо-параметри визначають достовірність вимірювання — побітова точність, ексклюзивний режим захоплення та відтворення — читайте у Теорії роботи ▸ Аудіо бекенд.

Відкривається через Tools ▸ Preferences. Захоплення та відтворення генератора призупиняються на час відкритого діалогу, щоб зміни пристрою або формату не порвали активний потік; OK застосовує та зберігає зміни, Cancel скасовує. Діалог організовано у п'ять вкладок: Вигляд & поведінка, Аудіо, Осцилограф, FFT та Частотна характеристика.

Вигляд і поведінка

ПараметрПримітки
Tab orientation Top (звичайна горизонтальна панель вкладок) або Left (вертикальна бічна панель із великими кнопками-іконками). Застосовується одразу — перезапуск не потрібен.
Small icons in main tab Зменшує іконки на головній вкладці. Застосовується одразу.
UI font / UI font (bold) Шрифти для підписів осей, показників і таблиць вимірювань (жирний — для виділеного тексту в таблицях). Застосовується після натискання OK.

Аудіо

Вибір аудіо-бекенду, вхідних / вихідних пристроїв і форматів — параметри, що визначають побітову точність вимірювання (див. Теорія ▸ Аудіо бекенд).

ПараметрПримітки
Backend WASAPI (типово, ексклюзивний режим із побітовою точністю), WDM-KS (найнижчий рівень, найвища пропускна здатність захоплення) або JavaSound (переносний / єдиний варіант поза Windows). Зміна значення повторно перераховує списки пристроїв.
Input — Device / Sample rate / Bit depth Пристрій захоплення (ADC) та точна частота дискретизації & розрядність, з якими він відкривається. 16/24 біт при 48/96/192/384 kHz підтримуються майже повсюдно; непідтримувана комбінація повідомляється під час запуску захоплення.
Output — Device / Sample rate / Bit depth Пристрій відтворення (DAC) та формат, що використовуються генератором і вимірюванням частотної характеристики.
В ексклюзивному режимі пристрій допускає лише один застосунок одночасно. Якщо список порожній або пристрій не відкривається, закрийте інші аудіо-застосунки та повторно відкрийте діалог для оновлення переліку.

Частотна характеристика

ПараметрПримітки
Trace line width, px Товщина кривих амплітуди, фази, еталону RIAA і порівняльної кривої.
Maximal analyzed frequency, % Nyquist Обмежує верхню межу перегляду цією часткою від Найквіста (96–100 %); значення нижче 100 % обрізає смугу, де деконволюція розгортки спадає.
Compare smoothing window, 1/N octave Ширина ковзного середнього для кривої режиму порівняння (виміряне − еталон). N = 0 вимикає згладжування; менше N = ширше = більше згладжування (N ≈ 3 — широке, N ≈ 24 — легке).
Allow notch filter for industrial noise Якщо ввімкнено, інтерполює через кожну гармоніку обраної частоти мережі для усунення піків гуму з відображуваної характеристики без повторного вимірювання.
Industrial frequency Базова частота мережі (50 / 60 Hz), гармоніки якої режектуються, якщо прапорець вище встановлено.
Signal / Phase / Reference (RIAA) / Chart background colour Кольори кривих і фону для перегляду частотної характеристики.
