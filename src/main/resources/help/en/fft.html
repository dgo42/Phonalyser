FFT analyser

Real-time spectrum + harmonic-distortion / noise-floor measurements. The analyser reads samples from the same shared capture buffer the oscilloscope uses; pressing the FFT's red record button doesn't change the scope's recording state. Analysis cadence scales with overlap — at 87.5 % overlap the spectrum refreshes ~8× per FFT window.

Header buttons

Row of square buttons at the top-left of the spectrum canvas.

L / R channel pick

Toggle buttons that pick which channel feeds the analyser. The selected channel's button takes a coloured background. Switching the channel resets the averaging buffer (see Reset) so the new channel's spectrum starts clean.

Auto-Setup

Arrows-to-circle icon. Frames the spectrum around the detected fundamental: frequency range ±1 decade, magnitude top = fundamental + 10 dB, bottom = noise floor − 20 dB. Requires at least one published analysis.

Maximise

Arrows-from-circle icon (the inverse of Auto-Setup). Sets the widest sensible view: frequency 0 .. Nyquist, top = +20 dB, bottom = noise floor − 30 dB. The top is signal-independent so the visible range stays stable even when the signal changes. Works with or without a current analysis (falls back to default bottom when no result yet).

Distortion table

Chart-column icon. Shows / hides the THD info table overlay in the top-left of the canvas. When the table is extracted to the external window (see below) this toggle still works — it controls whether the table is visible at all.

Reset averaging buffer

Red rotate-left icon. Wipes the averaging buffer + retained spectrum and rearms the gauge. The next analysis starts from zero — and clamps to fresh samples only, so old pre-reset audio doesn't bleed into the new average.

External-window toggle

Window-restore icon. Pops the THD info table out into a separate tool window so it can sit alongside the spectrum without occupying canvas space. Closing the tool window returns the table to the main view.

Plot region

Spectrum trace

Polyline of the FFT magnitude across the visible frequency range. Trace colour, line width, and chart background are configurable in Preferences → FFT. At high zoom levels the trace stays continuous: per-pixel envelopes collapse multiple bins into one vertical line, with adjacent known columns connected by a polyline segment.

Harmonic dots

Red dots painted at the detected fundamental and harmonic frequencies. Their diameter is configurable. When the manual-fundamental override is enabled, only the fundamental dot lifts to the override value; the rest stay on the calibrated dBV scale.

Distortion-band dimming

Light-grey overlay covering frequencies outside the active distortion HP / distortion LP window. Shows at a glance which bands are excluded from the SNR / THD+N integration.

Crosshair

Hovering shows a dashed grey cross at the pointer. The floating readout near the cursor lists the frequency (f = …) and the magnitude at that bin in the active unit (|m| = …). Suppressed when the pointer is over a header button.

Horizontal scrollbar (frequency)

Pans/zooms the visible frequency range. Drag the thumb to pan; resize the thumb to zoom. The thumb width reflects the visible fraction of the full range.

Vertical scrollbar (magnitude)

Same behaviour for the magnitude axis. Ctrl+wheel inside the canvas zooms the magnitude axis around the cursor; the scrollbar updates accordingly.

THD info table

Compact overlay in the top-left of the canvas (or in the external tool window when extracted). All values use the current magnitude unit. Hidden by the distortion-table toggle.

Header row

Bold first row: fundamental frequency (4-decimal Hz), dBFS, dBV. The dBV column uses the manual override when set; otherwise the ADC calibration anchor.

Span row

Span: 900 .. 9500 Hz — the band over which SNR / THD+N integrate. Span: full when both distortion bounds are disabled.

Clock-difference row (ΔF / Δosc)

ΔF: +0.001 Hz (+0.97 ppm) Δosc: +23.79 Hz @ 24.576 MHz. Shows the measured fundamental's deviation from the expected value (from the generator) — both in absolute Hz and ppm, and translated back to the master oscillator if the sample rate is a recognised 44.1 / 48 kHz family. Visible only when "Get fundamental from generator" is on AND the generator is actually running.

N+D (noise + distortion)

Total non-fundamental power, A-weighted. The "A" suffix flags the A-weighting.

N (noise-only)

Noise power with the detected harmonics removed from the band.

SNR

Signal-to-noise ratio in dB. Defined against the noise-only band (above).

THD H2..N

Total harmonic distortion as a percentage of the fundamental, summed over harmonics 2 through N (where N = Max harmonic for THD).

THD+N

Total harmonic distortion + noise. Same numerator as N+D, normalised to the fundamental.

ENOB

Effective number of bits = (SINAD − 1.76) / 6.02. Reflects the ADC + capture chain's combined dynamic range.

Harmonic rows (H2 .. HN)

Two harmonics per row. Each entry shows the harmonic's dBV level and its percentage of the fundamental. Number of rows is governed by Max harmonic to calculate.

Top toolbar (above the tab strip)

Fill percent

Small percentage label (e.g. 100%) showing how much of the next FFT window has accumulated since the previous analysis. Reaches 100 % at the moment the analyser fires.

Averages count

e.g. 32 average(s). Number of frames currently contributing to the displayed average. Caps at the configured Averages in fixed-window mode; keeps climbing in "forever" mode.

Magnitude unit

Combo at the top-right with four choices: V, V/√Hz, dBV, dBFS. Switching unit converts the axis labels and scrollbar bounds in place — the visible signal stays at the same screen Y.

Record button

Red round toggle, anchored to the right end of the toolbar row. Pressing it acquires a reference on the shared capture device and starts the FFT worker. The scope's record state is independent — both panes can record together. The button stays visible even when the toolbar tabs are collapsed.

Toolbar tabs

FFT settings tab

FFT length

Power-of-two combo from 8 k to 4 M. Bigger windows give finer bin resolution but slower update rate and more memory. 1 M at 384 kHz gives ~0.37 Hz/bin.

Window

Combo: Rectangular, Hann, Blackman-Harris 4, Blackman-Harris 7, Flat-top, Dolph-Chebyshev 150, Dolph-Chebyshev 200. Each trades main-lobe width against side-lobe level. Use Hann or Blackman-Harris for general work; Flat-top for accurate amplitude readout of an isolated tone; the DC150 / DC200 variants for very deep side-lobe suppression.

Overlap

0 %, 50 %, 75 %, 87.5 %, 92.75 %. Increases the update rate at the cost of CPU — successive analyses share that fraction of samples. 87.5 % is a common sweet spot.

Averages

Step selector — 2, 4, 8, 16, 32, ∞ (forever). Number of overlapped frames averaged per published spectrum. Higher values reduce noise variance but slow response. "Forever" accumulates indefinitely — combine with Stop after to cap.

Stop after N

Checkbox + numeric field. When the checkbox is on, the analyser pauses after N analyses (Forever averages mode only). Reset re-arms it.

Get fundamental from generator

When on, the analyser locks onto the generator's current frequency instead of auto-detecting the loudest peak. Lets you measure deeply notched fundamentals where a louder mains spur would otherwise win the detection. Automatically ignored when the generator isn't running.

Logarithmic frequency axis

Frequency axis switches between log and linear. Log is traditional for audio (constant pixel-per-octave); linear is useful for spotting evenly-spaced spurs.

THD settings tab

Enable distortion HP + frequency

Checkbox + frequency field. Sets the lower bound of the band used for SNR / N+D integration. Below this frequency, noise and spurs are excluded — useful for ignoring mains hum and DC-near artifacts.

Enable distortion LP + frequency

Same, for the upper bound. Excludes high-frequency noise above the band of interest (e.g. above 20 kHz for audio).

Manual fundamental

Checkbox + value + unit. When on, the analyser treats this value as the fundamental's TRUE dBV instead of inferring it from the calibrated ADC. The chart's fundamental dot and the THD-table header dBV column shift to this value; other bins and the harmonic-row dBV values stay on the calibrated scale. Useful when an external twin-T notch attenuates the fundamental but you know its true level.

Max harmonic for THD

Numeric (2 .. 9). Upper bound of the harmonics summed into the THD ratio. THD H2..N uses harmonics 2 through this number.

Max harmonic to calculate

Numeric (≥ Max harmonic for THD). How many harmonics to detect and list in the harmonic rows — independent of how many feed the THD ratio.

Coherent averaging

Checkbox. When on, complex spectra are averaged (so random- phase noise cancels across frames, lowering the noise floor ~3 dB per doubling of frame count). When off, power spectra are averaged (noise variance reduces but the floor doesn't drop). Coherent requires phase-stable signals — sweeps and impulse responses use power averaging instead.

Presets tab

Editable combo + Save / Load / Delete buttons. Save captures every FFT setting plus every THD setting plus the channel, magnitude unit, freq/mag axis ranges and log/lin flag. Load restores them. Delete asks for confirmation.

Utility tab

Screenshot

Camera icon. Renders the FFT pane to a PNG at a chosen resolution. Same offscreen-shell trick the oscilloscope uses, so the snapshot is a real layout (no bitmap scaling).

ADC calibration

Crosshair icon. Opens the same calibration dialog the scope's Utility tab uses.

Save to… tab

Writes the current spectrum to a CSV. Path field + browse + Save. Format: frequency_hz;magnitude_dBV;phase_deg, one row per bin.

Load from… tab

Reads a previously-saved CSV and renders it as a static spectrum overlay. Useful for comparing measurements taken at different times or under different conditions.

Tab tiles

Each tab's label has a row of small pill-shaped tiles below it showing the live state of the tab's main controls. Tiles are passive but their tooltips describe the value.

FFT-length tile

Short form on the FFT settings tab — e.g. 1M, 64k.

Window tile

3-letter abbreviation — RECT, HANN, BH4, BH7, FLAT, DC150, DC200.

Overlap tile

Percentage (87.5%).

Averages tile

Number with × suffix (32×) or ∞ in forever mode.

Coherent-averaging tile

coh when coherent averaging is on, inc when incoherent.

Distortion-range tile

On the THD settings tab. Shows the active band (900-10k) when both bounds are enabled, or a single bound (>900 / <10k) when only one is enabled. Absent when neither is set.

Max-harmonic tile

H9 — the max harmonic for the THD ratio.

Manual-fundamental tile

manF. Visible only when the manual-fundamental override is enabled.
