Layout

The main window is split into a Generator pane on the left (its width is adjustable — drag the sash, or set it in Preferences) and a vertical sash between Oscilloscope (top) and FFT analyser (bottom) on the right. Each pane can be collapsed to its title strip by double-clicking the title bar; double-clicking again restores it.

Each module has its own toolbar — a settings-tab strip below the main view with tabs for the per-module settings. The tab strip can be collapsed in turn (double-click the strip) to reclaim vertical space for the visible chart.

Conventions

  • Step selectors — fields like V/div, t/div, FFT length have small up/down arrows. The mouse wheel scrolls through standard values (1-2-5-10) and a manual typed value is accepted when it makes sense for the field.
  • Numeric fields — accept typed values plus mouse-wheel stepping. The wheel step is per-field (5 % for frequency, ±1 unit for harmonic numbers, etc.).
  • Tab tiles — the small badges painted under each tab label show the active value for that tab's most important controls at a glance, even when the tab body is collapsed.
  • Crosshair — hovering inside a chart shows a dashed cross with a floating readout box near the cursor.
